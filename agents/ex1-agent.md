# Agent Rules: Laboratory 06 — Exercise 1 (Simple Example, Single Main)

## Purpose
- Entry-level exercise for familiarization with the laboratory format and automatic testing.
- Designed to be approachable, with clear requirements and a single Main.java.

## Structure
- One Main.java file.
- One or more simple classes (e.g., Angajat).
- Input and output are explicitly specified in the Readme.
- No sub-parts; the whole exercise is solved in a single flow.

## Writing Style
- Step-by-step instructions.
- Explicit input/output examples for each requirement.
- Hints for standard Java features (e.g., Comparable, Comparator).
- No advanced features or edge cases.

## Grading
- Fully automatic, pass/fail per part (if split).
- No bonus points.
- All students should be able to pass if they follow the instructions.

## Typical Requirements
- Implement a simple class with a few fields.
- Implement Comparable for natural sorting.
- Implement at least one alternative Comparator.
- Read input, process, and print output in the required format.

- See laboratory06/exercise1/Readme.md for a canonical example.
