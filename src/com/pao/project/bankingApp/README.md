# Aplicatie Bancara

## 1.1 Actiuni posibile in sistem

1. Inregistreaza un client nou in sistem
2. Efectueaza un transfer bancar intre doua conturi
3. Plateste cu cardul la un comerciant
4. Aplica pentru un credit si achita rate lunare
5. Deschide un depozit la termen
6. Achizitioneaza o polita de asigurare
7. Genereaza un extras de cont pentru o perioada data
8. Detecteaza tranzactiile suspecte ale unui cont
9. Afiseaza cheltuielile unui client grupate pe categorii
10. Calculeaza averea neta a unui client
11. Calculeaza gradul de utilizare a cardurilor de credit
12. Identifica clientii cu risc ridicat
13. Afiseaza top N tranzactii dupa valoare
14. Grupeaza toate tranzactiile din sistem pe categorii
15. Ruleaza procesarea de sfarsit de luna
16. Genereaza un raport financiar complet pentru un client
17. Sterge un client din sistem
