# Agent Rules: Laboratory 06 — Exercise 3 (Bonus, Advanced, No I/O Tests)

## Purpose
- Optional, advanced exercise for bonus points and deeper understanding.
- Allows students to participate in only 8 exercises and win an additional 0.5% to the final laboratory grade.
- Can help reach the maximum grade even if the grade is not rounded to 50%.

## Structure
- No fixed input/output or test files.
- Demonstration is done in Main.java, with clear comments and output.
- More open-ended, focusing on integration, edge cases, and advanced features.

## Writing Style
- Requirements focus on integration of multiple concepts from the laboratory.
- Edge cases and error handling are explicitly required.
- No automatic testing; manual review is expected.
- Output and demonstration should be clear and well-commented.

## Grading
- Manual review.
- Bonus points (0.5%) awarded if completed and demonstrated correctly.
- Can be used to reach maximum grade even if the grade is not rounded.

## Typical Requirements
- Integrate and extend features from previous exercises.
- Demonstrate advanced use of interfaces, error handling, and design patterns.
- Handle and demonstrate edge cases (e.g., invalid input, unsupported operations).
- Output should be clear and demonstrate all required features.
